# News / Changelog

All notable changes to this project are documented in this file.

## v1.0.0 — 2026-04-24

### Added

- Initial release of the `ontoverse-kg-choreographer` package.
- Ontoverse pipeline: Zotero bibliography to Neo4j knowledge graph (topics, paper clones, similarity).
- Biomedical named entity recognition using scispaCy models and UMLS CUI integration.
- Paper similarity scoring from shared CUIs and Neo4j relationship population.
- Example configuration and documentation (`README.md`).
- Apache 2.0 license, `AUTHORS.md`, and `CONTRIBUTING.md`.
